Overview of Ymodem

Ymodem is a simple file transfer protocol that supports basic error detection methods to ensure that information is not lost or corrupted during a transfer. Ymodem is based on Xmodem and is essentially the same except that Ymodem sends the file's name, size, and timestamp before actually transferring the file.

The SecureCRT Ymodem implementation supports both 128-byte and Ymodem-1K (1024-byte) packet sizes.

Unlike Xmodem, Ymodem supports multiple-file (“batch”) transfers.
