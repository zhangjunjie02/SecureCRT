Keyboard Accelerators

This page lists keyboard accelerators (shortcut keys) that are supported in SecureCRT.  The accelerators are as follows:

•    CTRL+TAB takes you through multiple SecureCRT session tabs.

•    CTRL+SHIFT+TAB cycles backwards through the sessions.

•    COMMAND+W closes the active tab.

•    COMMAND+1 (2, 3, 4, 5, 6, 7, 8, 9, & 0) allows you to jump directly to one of the first ten tabs.

•    ALT+A re-establishes a connection on all currently disconnected sessions.

•    COMMAND+T opens the Connect in Tab/Tile dialog.

•    COMMAND+O opens the Connect dialog.

•    ALT+COMMAND+G switches the focus between the Command window and the active session.

•    ALT+I switches focus to the search bar in the Session Manager or Connect dialog.

•    ALT+M switches the focus between the Session Manager and the active session.

•    ALT+N switches the focus to the Session Manager or the Connect dialog where, if pressed again, ALT+N will open a new session.

•    COMMAND+SHIFT+P opens an SFTP tab using the active tab's session.

•     COMMAND+P prints the screen.

•    COMMAND+K opens the Quick Connect dialog.

•    ALT+R switches the focus between the connect bar and the active session.

For the advanced user, the Table of Keymap Functions lists keyboard menu functions that you can link to keystrokes using the Keymap Editor.
