The File Transfer/Filters category of the Global Options dialog allows you to configure filters to be used in file transfer operations.

Filters

This window displays the filters that SecureFX can use in synchronize operations.

To create a new filter, click on the Add... button to open a blank Filter Properties dialog. To modify and existing filter, select that filter and click on the Edit... button to open that filter's Filter Properties dialog. To remove an existing filter, select that filter and click on the Delete button.
